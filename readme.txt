== Description ==

Better Google Adsense is focused on easily displaying [Google Auto Ads](https://adsense.googleblog.com/2018/02/introducing-adsense-auto-ads.html) on your WordPress site. Choose to display ads on specific pages, categories, custom post types.

 Features include

* Drop in your Google Adsense code
* Enable ads on entire site
* Enable ads on specific WordPress pages
* Enable ads on specific WordPress post categories
* Enable ads on a custom post type


You will need to get your Auto Ads code from your [Google Adsense](https://www.google.com.au/adsense/start/) account.

Read more about [Adsense Auto Ads](https://adsense.googleblog.com/2018/02/introducing-adsense-auto-ads.html)

If you have any questions just let me know!


== Installation ==
Better Google Adsense plugin Installation Instructions:


1. Upload the Better Google Adsense plugin to your /wp-content/plugins/ directory or through the Plugin admin section under "add new".,
2. Activate the plugin through the ‘Plugins’ menu in WordPress.,
3. Configure your settings.
4. That's it!
